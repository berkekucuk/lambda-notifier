# Lambda Notifier

Lambda Notifier is a push notification service for mma-app.

It runs on AWS Lambda and is containerized with Docker.

## What It Does

- Sends push notifications when a fight result is finalized.
- Sends push notifications when the next fight is starting.

## Deployment

- Deployed on AWS Lambda.
- Built and packaged using Docker.

## Tech Stack

- Python
- Firebase Cloud Messaging (FCM)
- Supabase (REST API)
- AWS Lambda
- Docker
